# Dice Rolling 

This program will first ask user to select type of dice he/she wants to roll then it will ask the number of dice he/she wants to role and according to that it will print the results.


For Example:-
------------

Please enter the type of die you want to roll
(d6 or d20): d6

Enter the number of die you want to roll: 5

rolling d6s

[5, 4, 2, 1, 5]

Please enter the type of die you want to roll
(d6 or d20): d20

Enter the number of die you want to roll: 5

rolling d20s

[5, 1, 13, 19, 16]

Implementation:-
---------------

Random — Generate pseudo-random numbers

Source code: [Lib/random.py](https://hg.python.org/cpython/file/2.7/Lib/random.py)

This module implements pseudo-random number generators for various distributions.

For integers, uniform selection from a range. For sequences, uniform selection of a random element, a function to generate a random permutation of a list in-place, and a function for random sampling without replacement.
